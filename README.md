# Settle Up Website

A minimal, professional single-page website for Settle Up LLC with a Ferrari-inspired design.

## Features

- Single-scroll layout with smooth scrolling navigation
- Ferrari-inspired color palette (deep red #C00000, white, black, gray tones)
- Fully responsive, mobile-first design
- Smooth fade-in animations on scroll
- Semantic HTML5 structure with comprehensive meta tags
- Clean, minimal JavaScript for enhanced UX

## Files

- `index.html` - Main HTML structure
- `style.css` - All styling with CSS custom properties
- `script.js` - Smooth scrolling and scroll animations
- `favicon.svg` - Simple SVG favicon

## Deployment

### Cloudflare Pages

1. Push this repository to GitHub
2. Log in to [Cloudflare Dashboard](https://dash.cloudflare.com)
3. Go to Pages > Create a project
4. Connect your GitHub repository
5. Configure build settings:
   - Build command: (leave empty)
   - Build output directory: `/`
6. Deploy

### GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Under "Source", select the branch (e.g., `main`)
4. Select root `/` as the folder
5. Save and wait for deployment

## Local Development

Simply open `index.html` in your web browser. No build process required.

## Browser Support

Works in all modern browsers with support for:
- CSS Grid
- CSS Custom Properties
- Intersection Observer API
- Smooth scrolling
